GET station

Get the gas price (submitted by an Alpine user) at a particular gas station.

Resource

http://dev5:8888/station/99561


HTTP Request: Headers

The request method is GET so there is no body that needs to be described.
Only the basic headers are sent.
For example:

GET /station/99561 HTTP/1.1
Host: dev5:8888

HTTP Request: Headers: Request Method Parameters

id
(required)		 The objectId returned when the data was originally persisted to the database.

Example Request

http://dev5:8888/station/99561
