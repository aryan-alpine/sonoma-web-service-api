GET stations

Get a list of gas prices at gas stations near a particular GPS location.

Resource

http://dev5:8888/stations


HTTP Request: Headers

The request method is GET so there is no body that needs to be described.
Only the basic headers are sent.
For example:

GET /stations/radius/37.373596/-122.020500/2/pre/price/69r72p9zpe.json HTTP/1.1
Host: dev5:8888

HTTP Request: Headers: Request Method Parameters

longitude
(required)		 The longitude of the GPS location of interest.
latitude
(required)		 The latitude of the GPS location of interest.
distance
(required)		 The distance in miles from the specified gps location.
fuelType
(required)		 Choose one of {reg, mid, pre or diesel}
sortBy
(required)		 Choose one of {distance, price}
apiKey
(required)		 Use 69r72p9zpe.json

Example Request

http://dev5:8888/stations/radius/37.373596/-122.020500/2/pre/price/69r72p9zpe.json
