Alpine SpapGas REST API


The Alpine SnapGas Rest API is a subset of the Alpine REST API.

The purpose of this page is to describe how to use the SnapGas REST API calls to accomplish particular tasks.


Task 1: Get gas prices for all stations within a radius of a particular area.

GET stations

Note: Currently this service requires you to specify the type of gas that you are interested in. It returns the price of that type of gas only for all of the station.
If you are interested in knowing the prices of all types of gas at a particular station, you must call the following service:

GET station


Task 2: Get images.

There are 3 types of images associated with a gas station.

  1. thumnailImage
  2. fullImage
  3. userImage
Full URLs to each of these images are returned for the station in both of these restful services:

GET stations

GET station

To actually *retrieve* the image, you simply need to GET the URL.
The image will be downloaded as "Content-type: image/jpeg" as if resided on the file system.
In reality, the image is stored in a database and is being retrieved via the following restful service:

GET image


Task 3: Update Gas prices

To update one or more gas prices for a particular gas station, call this service:

PUT station/[stationId]


Task 4: Update user images

To add a user images for a particular gas station, call the same service, with different parameters:

PUT station/[stationId]/[imageType]/[imageFormat]

Method Resource Description
