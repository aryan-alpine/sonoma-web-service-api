GET routeHighlights

Get a list of "highlights" (as [lat,lng] coordinates) along a route between 2 [lat,lng] coordinates.
Previously known as "CMHgetRouteHighlights"

Resource

http://dev5:8888/routeHighlights?start=40.5761237,-74.002361&end=40.826396,-73.940125


HTTP Request: Headers

The request method is GET so there is no body that needs to be described.
Only the basic headers are sent.
For example:

GET routeHighlights?start=40.5761237,-74.002361&end=40.826396,-73.940125 HTTP/1.1
Host: dev5:8888

HTTP Request: Headers: Request Method Parameters

start
(required)		 latitude & longitude of the starting point of the route
end
(required)		 latitude & longitude of the end point of the route

http://dev5:8888/routeHighlights?start=40.5761237,-74.002361&end=40.826396,-73.940125
