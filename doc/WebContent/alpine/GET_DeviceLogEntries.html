GET DeviceLogEntry

Retrieve up to 100 device log entries.

Resource

http://dev5:8888/DeviceLogEntries?app=PinNFind

Parameters

app
(optional)		 Which application is giving us information?

Example Request

http://localhost:8080/alpine/GetDeviceLogServlet?app=PinNFind
