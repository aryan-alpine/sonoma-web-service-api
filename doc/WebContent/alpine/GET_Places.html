GET Places

Retrieve the list of places associatied with a particular user.

Resource

http://dev5:8888/Places?userId=12345


HTTP Request: Headers

The request method is GET so there is no body that needs to be described.
Only the basic headers are sent.
For example:

GET /Places?userId=12345 HTTP/1.1
Host: dev5:8888

HTTP Request: Headers: Request Method Parameters

userId
(optional)		 Which user's list of places do you want?

Example Request

http://54.245.243.173/Places?userId=12345
