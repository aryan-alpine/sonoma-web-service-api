GET weather

GET weather Get the weather for a particular location.
Previously known as "/re/weather"

Resource

http://dev5:8888/weather?ts=1234&app=TEST&dev=TEST&lat=37.296233&lng=-121.944508&APIKEY=ff924797a14aaa3111f9bbda5d3344d978959

OLD

https://www.aosvapps.com/re/weather?ts=1234&app=TEST&dev=TEST&lat=37.29623303279466&lng=-121.94450855255127&APIKEY=ff924797a14aaa3111f9bbda5d3344d978959


HTTP Request: Headers

The request method is GET so there is no body that needs to be described.
Only the basic headers are sent.
For example:

GET weather?ts=1234&app=TEST&dev=TEST&lat=37.296233&lng=-121.944508&APIKEY=ff924797a14aaa3111f9bbda5d3344d978959 HTTP/1.1
Host: dev5:8888

HTTP Request: Headers: Request Method Parameters

lat (required) latitude of the location you want weather information about.
lng (required) longitude of the location you want weather information about.
API_KEY (optional) Special key to help ensure that only approved client's may utilize this service.
ts (optional)
app (optional)
dev (optional) ?

Example Request

http://dev5:8888/weather?ts=1234&app=TEST&dev=TEST&lat=37.296233&lng=-121.944508&APIKEY=ff924797a14aaa3111f9bbda5d3344d978959
